# Projet dev movile

- Visualisation des conf de la dev fest
- Prise de notes lors des conf

# Commandes

## Installer les dépendances

Se positionner à la racine du projet

```
npm start
```

## Lancer le serveur web

```
ionic serve
```

## Lancer avec l'émulateur android studio

```
npx cap sync
npx cap run android
```
